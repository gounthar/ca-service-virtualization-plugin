Chemin d'accès au fichier d'entrée qui sera chargé dans l'environnement VSE. Le fichier d'entrée peut être un fichier de demande/réponse (fichier ou ZIP), VSM/VSI, Swagger, RAML ou WADL.
Possibilités :

pour les fichiers dans l'espace de travail du job, vous pouvez spécifier :

  • chemin d'accès relatif au fichier d'entrée dans l'espace de travail du job (exemple : dir/test-req.txt)

pour les fichiers sur l'ordinateur DevTest, vous pouvez spécifier :

  • chemin d'accès absolu au fichier (exemple : fichier:///C:/QuelqueChose/test-req.txt)
  • http emplacement du fichier d'entrée (exemple : http://QuelqueChose/test-req.txt)
