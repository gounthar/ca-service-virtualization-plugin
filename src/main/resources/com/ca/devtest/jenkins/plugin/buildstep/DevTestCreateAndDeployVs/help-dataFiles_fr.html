Chemin d'accès au fichier de données qui sera chargé dans l'environnement VSE. L'entrée doit être un fichier de données dans l'un des types de contenu définis. dans le fichier <LISA_HOME>/contenttype.xml,
Possibilités :

pour les fichiers dans l'espace de travail du job, vous pouvez spécifier :

  • chemin d'accès relatif au fichier d'entrée dans l'espace de travail du job (exemple : dir/data.csv)

pour les fichiers sur l'ordinateur DevTest, vous pouvez spécifier :

  • chemin d'accès absolu au fichier (exemple : fichier:///C:/QuelqueChose/data.csv)
