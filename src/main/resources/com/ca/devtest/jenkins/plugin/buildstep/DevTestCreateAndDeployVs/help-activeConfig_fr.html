Chemin d'accès au fichier de configuration du projet actif qui sera chargé dans l'environnement VSE.
Possibilités :

pour les fichiers dans l'espace de travail du job, vous pouvez spécifier :

  • chemin d'accès relatif au fichier d'entrée dans l'espace de travail du job (exemple : dir/project.config)

pour les fichiers sur l'ordinateur DevTest, vous pouvez spécifier :

  • chemin d'accès absolu au fichier (exemple : fichier:///C:/QuelqueChose/project.config)
